## 课程介绍

### 安卓逆向课程 - 2999元

**适合**：零基础 / Java开发转型 / 安全从业者想进阶

**学完你能掌握**：
- Hook框架：Frida、LSPosed开发与实战

- SO层逆向：ARM汇编、IDA Pro、算法还原
- 高级技术：OLLVM反混淆、白盒AES、eBPF
- 专项突破：Flutter逆向、Protobuf分析
